# offensive-notes 🏹💻

### Overview
This repository serves as a centralized technical repository for offensive security research. It contains detailed walkthroughs for Hack The Box (HTB) machines, structured Red Team mission notes, and custom exploit documentation developed during various research engagements.

### Repository Structure
- **`htb/`**: Step-by-step technical guides and reproduction steps for Hack The Box platforms.
- **`redteam/`**: Historical Red Team mission logs, artifacts, and technical reports (clean of personal documentation).
- **`bugbounty/`**: Research data, testing methodologies, and high-impact findings for bug bounty engagements.
- **`exploits/`**: Custom exploit scripts and vulnerability research proof-of-concepts.

### Example Usage
To browse documentation for a specific HTB machine:
```bash
ls htb/<machine_name>/
cat htb/<machine_name>/walkthrough.md
```
To find custom exploit code for a particular vulnerability class:
```bash
ls exploits/
```
