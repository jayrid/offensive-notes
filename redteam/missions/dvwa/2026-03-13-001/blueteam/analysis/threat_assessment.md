2. **Scanning:** Port 80 was identified as the primary attack vector. Directory enumeration revealed a sensitive configuration backup file (/config/config.inc.php.bak).
3. **Enumeration:** Between 02:16Z and 02:21Z, the attacker systematically confirmed three exploit-ready endpoints: SQLi on /vulnerabilities/sqli/, CMDi on /vulnerabilities/exec/, and XSS on /vulnerabilities/xss_r/.

   - **Sensitive Data Disclosure:** Exfiltrated database user credentials (app/vulnerables) from the exposed config.inc.php.bak file.
   - **XSS:** Demonstrated reflected cross-site scripting on the session cookie.
5. **Post-Exploitation:** The attacker confirmed the success of the exploits and finalized the exfiltration before mission completion at 02:45Z.
